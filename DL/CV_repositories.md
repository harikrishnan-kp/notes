# Low Light Image enhancement repos

- LLIE resource list: https://github.com/zhihongz/awesome-low-light-image-enhancement

1. Semantic-Guided-Low-Light-Image-Enhancement: https://github.com/ShenZheng2000/Semantic-Guided-Low-Light-Image-Enhancement
2. Zero-DCE (Zero-Reference Deep Curve Estimation) for low-light image enhancement
- Research paper: https://arxiv.org/pdf/2103.00860
- Repos:
    - original implementation of ZERO-DCE (pytorch): https://github.com/Li-Chongyi/Zero-DCE
    - TF implementation: https://github.com/tuvovan/Zero_DCE_TF
    - keras implementation: https://github.com/keras-team/keras-io/blob/master/examples/vision/zero_dce.py
3. Zero-DCE++ (Accelerated and light version of Zero-DCE): https://github.com/Thehunk1206/Zero-DCE?tab=readme-ov-file
4. MIRNet
- reasearch paper: https://arxiv.org/pdf/2003.06792v2.pdf
- Repos
    - TF implementation: https://github.com/soumik12345/MIRNet
    - TFLite variant: https://github.com/sayakpaul/MIRNet-TFLite-TRT
    - Keras : https://github.com/keras-team/keras-io/blob/master/examples/vision/mirnet.py
5. SCI
- Repos:
